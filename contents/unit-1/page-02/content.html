USE THE ADDRESS, MAX, AND ROW FUNCTIONS TO DETERMINE THE LAST CELL USED

With this tip, we learn the definition of an array formula. Here, we want to determine the last cell used in a range and shade it. Combine the ADDRESS, MAX, and ROW functions as described below to get the desired result.

To determine the last cell used in a range and shade it:

  1. In column A, list any kind of numbers.

  2. Select cell B2 and type the following array formula: =ADDRESS (MAX((A2:A100<>"")∗ROW(A2:A100)),1).

  3. Press < Enter >.

  4. Select cells A2:A11.

  5. From the Home tab, go to the Styles bar and click on Conditional Formatting.

  6. Choose New Rule.

  7. In the Select a Rule Type dialog, select Use a formula to deter- mine which cells to format.

  8. In the Edit box, type the following formula: =ADDRESS(ROW(),1)=$B$2.

  9. Click Format, select a color from the Fill tab, and click OK.

  10. Click OK.
